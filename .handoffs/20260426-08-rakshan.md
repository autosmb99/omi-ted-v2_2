## Key decisions
- **Write tool truncation**: Files >~90 lines get truncated silently. Use bash heredoc for anything longer.
  Already hit this on stats.py and globals.css — both fixed via bash.
- **Gold-only tokens**: Readiness thresholds are 20k (min) and 100k (solid) for gold-only.
  Old 50k/200k thresholds were for blended data including auto-translated.
- **Term consistency**: Multiple translations for the same Telugu word are NOT always wrong
  (e.g. "ప్రభువు" → "Lord" or "Master" depending on context). Dashboard flags them for human review,
  not automatic correction.
- **Multiple meanings / glossary**: User asked about this. Deferred — they said "maybe later".
  When ready: add `GlossaryTerm` endpoints (model already exists in models.py) + editor hints.

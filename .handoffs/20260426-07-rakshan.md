### API Keys Settings ✓
- `backend/routers/settings_api.py` — `GET/POST /api/v1/settings`
  - GET: returns which keys are set (boolean, never exposes values)
  - POST: writes to `backend/.env`, also updates `os.environ` live (no restart needed)
- `frontend/components/SettingsModal.tsx` — click ⚙ in nav to open
  - Shows "✓ Set" / "Not set" badge per key
  - Write-only: keys never shown back after saving
  - Saves once, persists forever to disk

### To set API keys
1. Click ⚙ top-right in the UI
2. Paste OpenRouter key (get from https://openrouter.ai/keys)
3. Paste Sarvam key if using Sarvam (https://dashboard.sarvam.ai)
4. Click "Save keys" — done, no restart, no terminal

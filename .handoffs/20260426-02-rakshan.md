## Step A — yt-dlp pre-flight (NOT done — needs Windows + browser)

This must still be run from PowerShell on your Windows machine. The Cowork Linux sandbox can't reach youtube.com (egress proxy blocks it) and can't read Windows browser cookie DBs.

```powershell
# Close all browsers first (yt-dlp needs exclusive read on the cookie DB)
cd C:\Users\kawin\Projects\omi-ted-v2\backend
.\venv\Scripts\Activate.ps1
python scripts\validate_transcripts_ytdlp.py
```

The script auto-cascades: Edge → Firefox → Chromium → Chrome → Brave.

If all browsers fail (App-Bound Encryption), manual cookies.txt fallback:
1. Install Chrome extension "Get cookies.txt LOCALLY"
2. Sign into youtube.com, click Export
3. Save to `C:\Users\kawin\Projects\omi-ted-v2\cookies.txt`
4. `$env:YTDLP_COOKIES_FILE='C:\Users\kawin\Projects\omi-ted-v2\cookies.txt'`
5. Re-run the script

Decision gate:
- **≥70% videos return Telugu auto-captions** → record number, proceed to M1 design

## Notes for Next Session
- **Python version**: must use Python 3.12 venv. Python 3.14 (the Windows Store default) fails to install pydantic-core and uvicorn.
- **Activating venv**: always `.\venv\Scripts\Activate.ps1` first. Check `(venv)` appears in prompt.
- **YTDLP_COOKIES_FILE**: set in every new PowerShell window before starting uvicorn, or add to `backend/.env` so python-dotenv picks it up automatically.
- **curl in PowerShell**: use `Invoke-WebRequest` not `curl` (PowerShell alias behaves differently). For POST with JSON: `-Method POST -ContentType "application/json" -Body '{"key": "val"}'`
- **`asyncpg` is commented out** in requirements.txt ‚Äî don't uncomment until Railway deployment.
- **en_auto quality**: YouTube's auto-translation is present but rough (good enough for M3 editor where humans correct it). M5 will replace with Sarvam/Google translation.
- **Stale .pyc on Windows mount**: not an issue on Windows (write access is fine). Only affects the Cowork Linux sandbox. On Windows, `--reload` picks up changes normally.
- **M1 idempotency**: `POST /api/v1/ingest/` returns `already_fetched` if the video is already done. To force re-ingest, delete the video row from dev.db and re-POST.
- **cookies.txt expiry**: YouTube session cookies expire. If ingest starts failing with auth errors, re-export cookies.txt from the browser.
